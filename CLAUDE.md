## 1. 프로젝트 개요

- **한 줄 정의**: 동의 기반으로 이력서를 수집한 인재를, **상위 대학 / 전 직장을 앞세워** 보여주는 프리미엄 인재 쇼케이스.
- **타겟 유저**: 한국 기업 인사담당자 (한국어 UI)
- **핵심 컨셉**: toptal.com 급의 프리미엄 UI/UX. "이 사람 어디 나왔고 어디 있었는지"가 3초 안에 꽂히는 카드.
- **원본과의 결정적 차이**:
  - 원본 = **익명**(학력/회사명 숨김). 이 프로젝트 = **동의 기반 노출**.
  - 인재에게 **개인정보 노출 동의**를 받고 이력서를 수집하는 플로우이므로, 상위 대학·전 직장을 당당히 강조하는 것이 핵심 가치.

## 2. 원본에서 걷어낸 것 (이 프로젝트에 없음)

아래 두 덩어리는 **삭제됨**. 다시 살리지 말 것. 관련 코드를 발견하면 잔재이므로 제거 대상.

- **AI 콜(음성 인터뷰)**: `app/interview/**`, `app/api/interview/**`, `lib/interview/**`, 음성 녹음(Recorder), TTS.
- **AI 이력서 스크리닝**: `app/api/screen-*`, `webhook/portfolio`, `lib/portfolio-screening.ts` 등.

## 3. 기술 스택 (원본 유지)

- **Frontend**: Next.js 14 (App Router) + TypeScript
- **Styling**: Tailwind CSS
- **DB/Auth**: Supabase (Postgres + RLS)
- **배포**: Vercel
- **폰트**: Pretendard Variable (CDN) — 한국어 UI 기준

## 4. 디자인 방향 — toptal 프리미엄 지향 (재정의 중)

> 원본은 Toss(TDS) 기반 플랫·미니멀이었다. 이 프로젝트는 **toptal 같은 프리미엄 감성**으로
> 개조 중이다. 정확한 색/타이포 토큰은 리디자인 진행하며 이 섹션에 확정해 나간다.
> 토큰이 확정되기 전까지는 임의로 색을 남발하지 말고, 변경 시 사용자와 확인한다.

- 학력/전 직장 **강조**가 카드의 시각적 1순위. (원본의 "학력·회사명 노출 금지" 규칙은 **폐기**)
- UI 카피는 **한국어**. (toptal은 레이아웃·톤만 참고, 영어 카피 이식 금지)
